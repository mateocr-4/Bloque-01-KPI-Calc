# Auditoría de Rentabilidad en Java (v1.0)

## Objetivo de negocio
Automatizar el cálculo de métricas financieras de marketing (ROI, CPL, Beneficio Neto) para eliminar el error humano en la gestión de cuentas de **Meta Ads**.

## Lógica de la herramienta
- **Cálculo de Comisión:** Aplica un 15% fijo sobre la inversión bruta.
- **Validación de Escalamiento:** El sistema determina si una campaña es apta para aumentar presupuesto basándose en el ROI (>50%) y la Tasa de Conversión (>2%).

## Tecnologías
- **Java 17/21**
- **Paradigma:** Programación Imperativa / Lógica de Negocio.
